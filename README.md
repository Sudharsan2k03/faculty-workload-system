# Faculty and Timetable Management System

## Project Overview
The Faculty and Timetable Management System is a full-stack web application designed to simplify academic scheduling and faculty workload management in educational institutions.

The system allows administrators to manage departments, faculty members, subjects, classrooms, and automatically generate timetable schedules while preventing scheduling conflicts.

This application improves efficiency by automating timetable generation, managing faculty workload distribution, and maintaining academic resources in a centralized system.

## System Modules

### Authentication Module
Handles secure login and protects system routes using authentication middleware.

### Department Management
Allows administrators to manage academic departments.

### Faculty Management
Stores faculty information and teaching workload details.

### Subject Management
Manages subject details and assignments.

### Classroom Management
Maintains classroom availability and capacity.

### Room Type Management
Defines room categories such as lecture halls or labs.

### Workload Management
Tracks and distributes teaching workload among faculty members.

### Timetable Management
Automatically generates timetable schedules while preventing conflicts.

### Activity Logs
Tracks administrative activities and system updates.

### Settings Module
Allows administrators to configure system settings related to scheduling.

## Installation Guide

1 Clone the Repository

git clone https://github.com/yourusername/faculty-timetable-management-system.git

2 Navigate to Project Folder

cd faculty-timetable-management-system

3 Install Backend Dependencies

cd backend

npm install

4 Install Frontend Dependencies

cd frontend
npm install

5 Run Backend Server

npm start

6 Run Frontend Application
npm run dev
